# 裁判報名與派遣系統（Umpire Management System）

本系統用於管理棒球賽事裁判之報名、取消、重新報名與派遣流程，
支援桌機與手機版（RWD），並提供管理層即時通知機制。

---

## 📂 文件目錄說明

- `doc/README_tech.md`  
  技術文件，提供系統架構、API、資料表與後端邏輯說明。

- `doc/README_user.md`  
  使用者操作手冊，提供裁判與管理者實際操作說明。

- `doc/README_v2.md`  
  v2 擴充藍圖，說明管理後台與未來功能規劃。

- `doc/system_tree.txt`  
  系統整體結構樹狀圖（文字版）。

- `doc/state_flow.txt`  
  裁判報名／取消狀態流程圖（文字版）。
