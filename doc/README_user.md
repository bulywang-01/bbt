# 裁判報名系統使用說明

本系統提供裁判線上報名、取消與查詢執勤賽事。

---

## 一、登入系統

1. 開啟系統網址
2. 輸入帳號密碼登入
3. 登入後會看到首頁公告與近期賽事

---

## 二、裁判報名

### 桌機操作
- 進入「裁判報名」
- 在表格中選擇站位（主審 / 一壘審等）
- 點擊「送出報名」

### 手機操作
- 進入裁判報名頁
- 每一場賽事以卡片顯示
- 點選「報名」按鈕
- 底部點擊「送出報名」

---

## 三、取消站位

- 只有「自己報名的站位」可以取消
- 點擊「取消」後確認即可

### 取消限制
- 比賽已過期：不可取消
- 比賽前一天（含）：不可取消
- 請聯絡裁判長處理臨時狀況

---

## 四、重新報名

- 若曾取消過該場賽事
- 可再次報名
- 系統會在送出後提醒「曾取消過」

---

## 五、通知說明

- 成功取消站位後
- 系統會自動通知：
  - 系統管理員
  - 裁判長
