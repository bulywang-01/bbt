# v2 管理後台擴充規劃

本階段以「不影響現有裁判流程」為原則，
逐步加入管理功能。

---

## 第一階段（高優先）

### admin_dashboard.html
- 賽事列表檢視
- 每場賽事裁判配置一覽
- 顯示缺額 / 已額滿狀態

### 管理者直接指派裁判
- 略過裁判報名
- 強制指定站位
- 記錄為 system_assigned

---

## 第二階段（中優先）

### 裁判管理
- 啟用 / 停用裁判帳號
- 設定裁判等級
- 搜尋裁判歷史執勤

### 賽事管理
- 新增 / 編輯 / 關閉賽事
- 設定 umpire_count
- 設定站位模式

---

## 第三階段（進階）

### 報表與匯出
- 匯出班表（CSV / PDF）
- 每月執勤統計

### 權限與稽核
- 操作紀錄 Log
- 取消原因紀錄
- 管理操作審計

---

## 設計原則

- v1 裁判流程「封版不動」
- v2 僅加功能，不破壞既有資料
- 所有異動留歷史紀錄
