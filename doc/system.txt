狀態流程圖
【尚未報名】
     │
     ▼
【送出報名】
     │
     ▼
【approved（有效報名）】
     │
     ├─（取消站位）
     │        │
     │        ▼
     │   【cancelled（已取消）】
     │        │
     │        └─ 可重新報名（新增一筆）
     │
     └─（比賽結束）
              │
              ▼
        【歷史紀錄（唯讀）】
